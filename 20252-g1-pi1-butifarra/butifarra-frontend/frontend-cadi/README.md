# React + Vite

## Flujo de autenticación

La pantalla de inicio de sesión incluye ahora un acceso directo al formulario de registro de Django. Tras el botón de "Ingresar" encontrarás el enlace **"Regístrate aquí"**, que abre la vista `/register/` en una pestaña nueva para que las personas usuarias puedan crear su cuenta sin abandonar el frontend.
